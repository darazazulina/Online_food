## Исходные данные

Набор данных [Online Food Dataset](https://www.kaggle.com/datasets/sudarshan24byte/online-food-dataset) размещен на kaggle.

Этот набор данных содержит информацию, собранную с онлайн-платформы заказа еды за определенный период времени. Он охватывает различные атрибуты, связанные с профессией, размером семьи, отзывами.

### Атрибуты таблицы

- Возраст: количество полных лет клиента;
- Пол: женский или мужской;
- Семейное положение: одинокий, в браке или неизвестно;
- Род занятий: выбиант занятости;
- Ежемесячный доход: доход клиента в месяц;
- Образовательная квалификация: образование клиента;
- Размер семьи: количество человек в семье;
- Широта: широта местонахождения клиента;
- Долгота: долгота местонахождения клиента;
- Индекс: индекс местонахождения клиента;
- Обратная связь: отзыв, предоставленный клиентом после получения заказа.

## Дашборд

Дашборд представлен на двух страницах. На одной из них статистика по демографическим показателям, а на другой географическое положение клиентов. В отчете использованы следующие элементы:

- карточка,
- линейчатая диаграмма с накоплением,
- гистограмма с накоплением,
- круговая диаграмма,
- карта,
- таблица.
